# Structive

Structure-driven template engine where structure defines meaning.

## Overview

Structive is a declarative template engine where structure itself carries meaning. It enables intuitive data binding and componentization while leveraging HTML structure.

### Getting Started

- [Quick start](docs/getting-started.md)
- [Error codes](docs/error-codes.md)

## Features

- **Structure-driven**: Declarative syntax based on HTML structure
- **Data Binding**: Simple yet powerful data binding capabilities
- **Web Components**: Full support for Web Components standard
- **Lightweight**: Fast and lightweight runtime

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Run tests with coverage
npm run test:coverage

# Lint
npm run lint

# Auto-fix lint issues
npm run lint:fix

# Build
npm run build

# Clean dist artifacts
npm run clean
```
