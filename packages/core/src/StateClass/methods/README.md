# StateClass Methods Overview

Low-level helpers for manipulating state class instances and indexes.

- `checkDependency.ts`: Determines whether a dependency requires update.
- `createListIndexes.ts`: Builds list index structures for repeaters.
- `getByRef.ts`: Resolves state values by ref metadata.
- `getContextListIndex.ts`: Fetches indexes relative to binding context.
- `getListIndex.ts`: Retrieves list index entries by name.
- `getListIndexesByRef.ts`: Collects all indexes tied to a ref.
- `setByRef.ts`: Writes state values given ref metadata.
- `setLoopContext.ts`: Updates loop contexts associated with state classes.
